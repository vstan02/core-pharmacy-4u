# Pharmacy 4U
> Online pharmacy for you

[![BCH compliance](https://bettercodehub.com/edge/badge/vstan02/pharmacy-4u?branch=master)](https://bettercodehub.com/)

### Build Setup
```bash
# install dependencies
npm install

# build styles
npm run styles

# build scripts
npm run scripts

# build images
npm run images

# build all assets
npm run assets

# build for production
npm run build

# run server (without build)
npm run dev

# run server (with build)
npm run start

# run tests
npm run test

# run linter
npm run lint
```
